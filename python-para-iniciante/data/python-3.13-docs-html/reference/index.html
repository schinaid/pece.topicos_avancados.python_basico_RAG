A Referência da Linguagem Python¶

Este manual de referência descreve a sintaxe e a “semântica central” da linguagem. É conciso, mas tenta ser exato e completo. A semântica dos tipos de objetos embutidos não essenciais e das funções e módulos embutidos é descrita em A Biblioteca Padrão do Python. Para uma introdução informal à linguagem, consulte O tutorial do Python. Para programadores em C ou C++, existem dois manuais adicionais: Estendendo e incorporando o interpretador Python descreve a imagem de alto nível de como escrever um módulo de extensão Python, e o Manual de referência da API Python/C descreve as interfaces disponíveis para programadores C/C++ em detalhes.
